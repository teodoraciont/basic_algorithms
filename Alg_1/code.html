Algoritmul pentru transformarea unei caracter in litera mare este: 

    caracter: numar intreg
    citeste litera
    daca caracter >= 'a' SI caracter <='z' atunci
        caracter = caracter - 32

Algoritmul pentru transformarea unei caracter in litera mica este: 

    caracter: numar intreg
    citeste litera
    daca caracter >= 'a' SI caracter <='z' atunci
        caracter = caracter + 32
